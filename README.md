# Internet-Through-Drones-in-Disaster-Prone-Areas-FYP-
This project focuses on the development and evaluation of a drone-based mobile ad-hoc network (MANET) designed to restore internet connectivity in disaster-recovery zones and remote, infrastructure-deprived environments. By deploying a swarm of autonomous, intelligent drone nodes, the system creates a resilient, self-healing mesh network capable of maintaining communication links despite high node mobility and frequent topology changes.

Core Objectives & Capabilities
Autonomous Mesh Networking: Implemented and configured OLSRv2 (Optimized Link State Routing Protocol) to enable seamless peer-to-peer communication, ensuring the network can autonomously discover routes and adapt to the constant movement of drones.

Resilience Engineering: Engineered the system to withstand challenging operational conditions, specifically mitigating high-latency links and intermittent signal loss to maintain high data throughput.

Performance Optimization: Conducted rigorous analysis of critical network metrics, including packet forwarding efficiency, link quality estimation, and routing table stability.

Simulation & Modeling: Leveraged advanced network simulation tools to model complex behaviors, providing data-driven insights into how node mobility patterns impact long-term connection persistence.

Technical Stack
Routing Protocol: OLSRv2

Network Domain: MANET / Mesh Networking

Key Focus Areas: Network Resilience, Routing Stability, Mobility Modeling
